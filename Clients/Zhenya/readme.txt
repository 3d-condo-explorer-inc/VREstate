При разработке использовался Eclipse Helios release 2 (http://eclipse.org).
GWT плагин для Eclipse (http://code.google.com/intl/uk-UA/eclipse/docs/getting_started.html)

Потом выбираем меню File=>Import=>General=>Existing projects into workspace.


В данной версии ещё много чего нет. Если успею, то до вечера скину новою версию.

Пути описаны в файле конфигурации (Options.java).
Содержимое папки "DATA" нужно скопировать в папку WebRoot сервера VREServer.
